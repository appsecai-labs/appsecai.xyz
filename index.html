AppSecAI

AI tools and experiments focused on improving application security workflows.

Open-source projects exploring how artificial intelligence can support secure software development.

Purpose

AppSecAI is a security engineering lab focused on experimenting with practical applications of artificial intelligence in application security.

The initiative explores how security can integrate more effectively into modern software development processes through automation and developer-focused tooling.

Current Projects

PR Security Agent

An AI-assisted security reviewer designed to analyze Pull Requests and identify potential vulnerabilities before code is merged.

The goal is to bring security feedback directly into the developer workflow and reduce manual security review effort.

View tool →

ThreatModel AI

An experimental project exploring how AI can assist with threat modeling by converting architecture descriptions into structured threat models.

The tool integrates with Threagile to generate threat analysis automatically from architecture definitions.
