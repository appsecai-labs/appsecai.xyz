Tools

AppSecAI develops experimental tools focused on improving application security workflows using automation and artificial intelligence.

PR Security Agent

PR Security Agent is an AI-assisted security reviewer designed to analyze Pull Requests and detect potential security vulnerabilities before code is merged.

The goal is to integrate security feedback directly into the developer workflow and reduce the time required for manual security reviews.

  • Analyzes code changes in Pull Requests
  • Detects potential injection vulnerabilities
  • Identifies hardcoded secrets and unsafe patterns
  • Posts security findings as PR comments

Status: Early development

Repository: github.com/appsecai-labs/pr-security-agent

ThreatModel AI

ThreatModel AI explores the use of artificial intelligence to assist with threat modeling by converting architecture descriptions into structured threat models.

The project integrates with Threagile to generate threat analysis based on system architecture definitions.

  • Generate threat models from architecture descriptions
  • Automate creation of Threagile YAML models
  • Support security architecture analysis

Status: Research / design phase
