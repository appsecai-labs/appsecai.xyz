PR Security Agent

An AI-powered assistant that reviews Pull Requests and detects potential security vulnerabilities.

Focus areas include:

  • Injection vulnerabilities
  • Hardcoded secrets
  • Unsafe coding patterns

Repository: https://github.com/appsecai-labs/pr-security-agent

ThreatModel AI

A threat modeling assistant that converts architecture descriptions into structured threat models using AI and Threagile.

The goal is to simplify threat modeling for engineering teams and make security architecture reviews easier.
