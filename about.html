About

AppSecAI

AppSecAI is a security engineering lab focused on exploring practical applications of artificial intelligence in application security.

The goal of AppSecAI is to experiment with tools and approaches that improve how security integrates with modern software development workflows.

Areas of exploration include secure code review automation, threat modeling, developer security tooling, and AI-assisted security analysis.

Focus Areas

  • AI-assisted secure code review
  • Threat modeling automation
  • Developer security workflows
  • Application security tooling

Open Source

Projects developed under AppSecAI are published as open-source tools. The intention is to experiment, learn, and share approaches that may help improve application security practices.

Current projects are available through the AppSecAI GitHub organization.
